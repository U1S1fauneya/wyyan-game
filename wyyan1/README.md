## 🎯 游戏机制

### 💖 生命值系统
- **初始生命值：** 3条 (❤️❤️❤️)
- **碰到敌人：** -1条生命 + 1秒无敌时间
- **收集爱心：** +1条生命（最多5条）
- **生命为0：** 游戏结束

### 🎮 手机操作方式
- **开始游戏: ** 触摸屏幕中的开始按钮开始
- **移动控制：** 拖拽主角进行水平移动射击
- **射击攻击：** 点击右下角射击按钮





### 🏆 得分系统
- **消灭敌人：** +10分 👿

- **碰撞到敌人:** 生命值-1

- **收集爱心：** +50分 💗
